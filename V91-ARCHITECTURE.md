 # Listener Integration Tests\n└── README.md\n```\n\n**Kern-Funktionalität:**\n\n```python\n# 1. Regeln evaluieren\naction = controller.evaluate_rules()\n# Mögliche Actions: scale_up_workers, scale_down_workers, activate_chaos_mode\n\n# 4. Ingest (06:00 UTC)\n   matrix_data.json → Schema Validation → Confidence Scoring (OPA)\n   \n2. Decision Tree\n   ├─ Confidence ≥0.85  → Auto-VALID → Sign → Release\n   ├─ 0.5–0.85          → Create PR → Verifier Review (SLA: 8h)\n   └─ <0.5              → Quarantine → Alert (P2)\n\n3. Signing\n   Hash (SHA256) → Cosign Sign → Transparency Log (Rekor)\n   \n4. Ledger\n   audit_log.jsonl ← Append-only, signiert, unveränderbar\n```\n\n**Confidence Scoring (OPA):**\n```rego\nConfidence = (\n  domain_allowed(entry)  * 0.4 +\n  valid_schema(entry)    * 0.2 +\n  date_valid(entry)      * 0.15 +\n  meta_present(entry)    * 0.15 +\n  recent_entry(entry)    * 0.1\n)\n```\n\n**SLA & Alerts:**\n| Alert | Condition | Action | SLA |\n|-------|-----------|--------|-----|\n| P1 | Hash Mismatch | Pager | 1h |\n| P2 | Source Down >6h | Chat | 6h |\n| P3 | PR Backlog >24h | Summary | 24h |\n\n**Start:**\n```bash\n# Secrets setup\ngh secret set COSIGN_KEY\ngh secret set COSIGN_PASSWORD\n\n# Manual trigger\ngh workflow run v91-audit-pipeline.yml\n```\n\n---\n\n### 3. **v91-interop-listener** (Federation & Quarantine)\n\n**Integriert in:** v91-core-system → `controller/v91_listener.py`\n\n**Zweck:** BFT Validierung, Event-Federation, Quarantäne-Enforcement\n\n**Kern-Funktionalität:**\n\n```python\n# Event-Listener-Loop\nwhile True:\n    for event_file in inbox:\n        event = load_json(event_file)\n        \n        # 1. Signatur prüfen\n        if not verify_signature(event):\n            quarantine_event(event, \"Invalid signature\")\n            continue\n        \n        # 2. Ledger-Integrität prüfen (prev_hash == last current_hash)\n        if not self_audit():\n            quarantine_event(event, \"Ledger chain broken\")\n            continue\n        \n        # 3. Commit mit Retry (exponential backoff)\n        success = commit_with_retry(event, max_retries=5)\n        \n        # 4. Inbox cleanup\n        if success:\n            remove(event_file)\n```\n\n**Quarantäne-Tabelle:**\n```sql\nCREATE TABLE v91_quarantine (\n  id INTEGER PRIMARY KEY,\n  component TEXT,\n  payload JSON,\n  reason TEXT,\n  timestamp DATETIME DEFAULT CURRENT_TIMESTAMP\n);\n```\n\n**Docker Compose Integration:**\n```yaml\nlistener:\n  build: .\n  command: python controller/v91_listener.py\n  volumes:\n    - ./data/inbox:/app/data/inbox\n    - ./data/v91_independent_state.db:/app/data/v91_independent_state.db\n  restart: always\n  healthcheck:\n    test: [\"CMD\", \"curl\", \"-f\", \"http://localhost:8000/health/v91-listener\"]\n    interval: 30s\n    timeout: 10s\n    retries: 3\n```\n\n**Test-Szenarios:**\n```bash\n# Valid Event (erscheint im Ledger)\necho
